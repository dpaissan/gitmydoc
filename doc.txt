Diff:
  git diff origin/master master

Restore a file:
  git co -- [file-name]

Create a new branch:
  git co -b [branch-name]
  git push origin [branch-name]

Tag:
  git add -A
  git commit -am "bla bla"
  git push origin [branch-name]
  git tag -a 1.10.1 -m "Version 1.10.1"
  git push origin [branch-name] --follow-tags

Merge:
  Resolve conflicts in a file:
    1. git mergetool [nomefile]
    2. ... VS starts ...
    3. ... change the file in VS ...
    4. ... save the file in VS ...
    5. ... close VS ...
    6. git commit
          
    Overwrite the file with the local version:
      git checkout --ours [file-name]
      git add [file-name]     # ⚠️⚠️TO VERIFY⚠️⚠️
          
    Overwrite the file with the remote version:
      git checkout --theirs [file-name]
      git add [file-name]     # ⚠️⚠️TO VERIFY⚠️⚠️
          
    Abort the merge phase if nothing has been committed:
      git merge --abort

Move uncommitted changes:
  1. git stash
  2. ... move to another branch ...
  3. git stash pop

Delete stash:
  git stash drop
          
Show all remote brances:
  git branch -r
          
Edit unpushed commit message:
  git commit --amend
    
Force-overwrite remote release branch with local release branch:
  git push origin +release:release

Move branch to a specific commit:
  git reset --hard <commit-id>
  git push --force origin <branch>

Move tag to a specific commit:
  git tag -d <tagname>                    # delete the old tag locally
  git push origin :refs/tags/<tagname>    # delete the old tag remotely
  git tag <tagname> <commit-id>
  git push origin <tagname>

Prevent merge to create '.orig' backup files:
  git config --global mergetool.keepBackup false

.gitconfig path:
  C:\Users\[username]\.gitconfig
